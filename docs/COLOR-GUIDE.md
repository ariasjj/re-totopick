## 🎯 색상 선택 팁

### 1. 사이트 목적에 맞게 선택
- **기업/공식**: 파란색, 회색
- **열정/스포츠**: 빨간색, 주황색
- **자연/건강**: 초록색
- **럭셔리**: 보라색, 검정색
- **창의/예술**: 보라색, 핑크색

### 2. 대비(Contrast) 확인
```css
/* 배경이 어두우면 글자는 밝게 */
--primary: 210 100% 20%;          /* 어두운 파랑 */
--primary-foreground: 0 0% 100%;  /* 흰색 글자 */

/* 배경이 밝으면 글자는 어둡게 */
--primary: 210 100% 90%;          /* 밝은 파랑 */
--primary-foreground: 0 0% 10%;   /* 검정 글자 */
```

### 3. 일관성 유지
- 메인 색상: 버튼, 링크, 강조
- 보조 색상: 배경, 카드
- 강조 색상: 알림, 특별 요소

## 🔢 HSL 색상 이해하기

```css
--primary: 210 100% 50%;
           ↑   ↑    ↑
         색조 채도 명도
```

### 색조 (Hue: 0-360)
- 0-60: 빨강 → 주황
- 60-120: 노랑 → 초록
- 120-180: 초록 → 청록
- 180-240: 파랑
- 240-300: 보라
- 300-360: 핑크 → 빨강

### 채도 (Saturation: 0-100%)
- 0%: 회색 (무채색)
- 50%: 중간
- 100%: 선명

### 명도 (Lightness: 0-100%)
- 0%: 검정
- 50%: 표준
- 100%: 흰색

## ⚠️ 주의사항

### 접근성 (Accessibility)
```css
/* ❌ 나쁜 예: 대비가 낮음 */
--primary: 60 100% 90%;           /* 밝은 노랑 */
--primary-foreground: 0 0% 100%;  /* 흰색 글자 → 안 보임! */

/* ✅ 좋은 예: 대비가 높음 */
--primary: 60 100% 40%;           /* 어두운 노랑 */
--primary-foreground: 0 0% 100%;  /* 흰색 글자 → 잘 보임! */
```

### 색맹 고려
- 빨강과 초록만으로 정보 전달 X
- 색상 + 아이콘/텍스트 함께 사용
